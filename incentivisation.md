Stakeholder Incentives TODO
Goal
Make the app feel like a time-saver and risk-reducer for teachers, not an extra admin step.

North Star
Teachers should be able to book or adjust a class room in under 30 seconds, with zero duplicate data entry.

Priority 1: Faculty Incentives (Most Important)
One-click Repeat Booking
Build weekly recurring booking in one action (entire semester/term), so teachers do not create each booking manually.

Timetable Auto-Sync
Import official timetable and pre-fill teacher schedules. Teachers should only edit exceptions, not enter base schedule.

Conflict-Free Smart Suggestions
If a room is unavailable, show 3 instant alternatives by:
capacity fit, nearest building, required facilities, closest time slot.

Emergency Fast Book Mode
Create a 2-click quick booking flow for urgent quizzes/tutorials with default preferences.

No Double Entry Integration
Integrate with existing college tools (ERP/LMS/Google Calendar) so bookings auto-appear where teachers already work.

Auto Notifications to Students
When teacher changes room, students get notification automatically. This removes manual WhatsApp/class group effort.

“My Usual Rooms” Presets
Allow teachers to save preferred room templates (capacity + projector + building).

Last-Minute Room Recovery
When booking fails, provide a fallback reserve queue with admin-assisted auto-allocation.

Personal Time Saved Dashboard
Show each teacher:
hours saved, conflicts avoided, successful auto-suggestions used.
This creates visible personal benefit.

Priority 2: Admin Incentives (Supports Faculty Adoption)
Department Adoption Scoreboard
Track adoption by department and identify low-usage areas for targeted onboarding.

SLA for Booking Resolution
Define response SLA for special requests (example: under 15 minutes in working hours).

Approval Burden Reduction
Auto-approve low-risk faculty bookings based on rules (trusted users, standard rooms, non-overlap).

Utilization-Based Recognition
Monthly reports for departments with highest efficient usage and lowest conflict rates.

Policy Automation
Encode rules (lab-only access, capacity restrictions, maintenance windows) to reduce manual intervention.

Priority 3: Student Incentives (Network Effect)
Live “Where is My Class” View
Students instantly see current room and last-minute changes.

Delay Prevention Alerts
Push alerts if room changes or class relocation happens.

QR Check-In Utility
Teacher gets quick attendance support; students get accurate room confirmation.

Empty Room Discovery
Allow controlled discovery of free rooms for study groups (with permissions).

Product and UX TODO (Make It Feel Easier Than Manual Work)
Under-30-Second Booking UX
Measure and optimize click count and form length.

Mobile-First Teacher Flow
Most urgent bookings happen on phone; prioritize mobile quick actions.

Smart Defaults
Auto-fill date/time based on timetable and current context.

“Book Again” Button
Repeat past booking in one tap.

Zero-Learning-Curve Onboarding
Interactive first-use checklist with preloaded sample booking.

Behavioral Incentive Layer
Faculty Priority Access
Frequent users get early access to premium rooms/time windows.

Reliability Badge
Show trusted faculty status for consistent proper usage.

Instant Confirmation Guarantee
For eligible requests, promise immediate confirmation instead of pending state.

Monthly “Time Saved” Email
Send personal summary to reinforce value and habit formation.

Adoption Experiment Backlog (Growth Loop)
A/B Test: Quick Book vs Standard Form
Success metric: booking completion rate and time-to-book.

A/B Test: Auto-Sync Onboarding vs Manual Start
Success metric: week-1 teacher activation.

Pilot Program by One Department
Run 2-week pilot, collect testimonials, then expand campus-wide.

Referral Loop for Faculty
If a teacher invites another teacher who becomes active, unlock priority credits.

“First 3 Bookings Assisted” Program
Offer concierge setup for first-time users to remove fear/friction.

Success Metrics (Track Weekly)
Teacher Activation Rate (first week) target: 60%+
Average Booking Time target: below 30 seconds
Repeat Usage (weekly active faculty) target: 50%+
Conflict Prevention Rate target: 95%+
Manual Admin Interventions target: down by 40%
Room Change Notification Delivery target: 99%+
